¿Qué necesito?

Antes de todo un ordenador o portatil, no hace falta que sea el mejor, con algo básico es suficiente.
Una vez obtenido el ordenador solamente debe descargarse la aplicación correspondiente dependiendo del lenguaje que quiera utilizar.
Por ejemplo, si desea utilizar Python, descarguese PyCharm, si desea escribir en Java puede descargarse IntelliJ IDEA y así con los demás lenguajes.
Si no quiere quedarse con un solo lenguaje puede probar a utilizar VSC (Visual Studio Code) y aunque disponga de pluggins o extensiones resulta un poco más limitado que los IDE (entorno de desarrollo integrado)

¿Como comenzar?

Primero debemos crear un nuevo proyecto con el nombre que queramos, pero preferiblemente Hello World, dentro de la aplicación donde deseamos hacer el Hello World.
Ahora enseñaré los pasos a seguir con los lenguajes más comunes y usados explicando la función de todo lo que se vaya escribiendo.

Python

Con Python es muy sencillo, lo único que debemos hacer es escribir 'print', lo que hace este comando es mostrar una cadena de texto en la terminal o pantalla, seguidamente ponemos entre paréntesis y comillas dobles el texto que queremos que aparezca por pantalla.
Los comandos en pantalla se verán de esta manera:

print("Hello, World!")

JavaScript

En este caso tambien es bastante sencillo, si quieres que se muestre en el terminal lo único que debes hacer es abrir la etiqueta script y dentro escribir console.log y luego entre parentesis y comillas dobles el texto que queremos que aparezca en el terminal. Se vería así:

< script >
      console.log("Hello, World!");
< /script >
Si quieres que el texto salga en una página web en vez de una terminal lo único que debes cambiar es el console.log por el document write, quedaría de la siguiente manera:
< script >
      document.write("Hello, World!");
< /script >

Java

En Java ya es un poquito más complicado que el resto de lenguajes explicados anteriormente, pero no es nada del otro mundo.
Lo primero que se debe de hacer es crear una clase con el nombre que queramos y decir que tipo de privacidad tendrá, normalmente las clases son públicas y se declaran con 'public class NombreDeLaClase' luego debemos llamar el método main escribiendo main y apretando el 'Enter', main es el método que se ejecuta cuando inicias un programa. Luego escribimos sout (System.out.println()) y entre los parentesis y comillas dobles escribimos 'Hello, World!', quedaría así:

public class HelloWorld {
public static void main(String[] args) {
System.out.println("Hello, World!");
    }
}

C++

En C++ hay diversas formas de mostrar Hello, World! por pantalla o terminal pero una de las opciones que considero más sencillas es la siguiente: Primero escogemos la biblioteca de entrada y salida, en este caso es <studio.h> luego decimos donde comienza la función principal del programa con int main, seguidamente indicamos que vamos a imprimir texto por pantalla con printf y entre los paréntesis y comillas dobles escribimos el texto que queremos que aparezca por pantalla.
El codigo escrito se ve de la siguiente manera:



#include < stdio.h >
        int main() {
                printf("Hello World!");
        }
